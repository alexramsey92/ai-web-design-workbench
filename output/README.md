# Output Directory

This directory contains HTML files generated by the AI Web Design Workbench.

## Usage

When you run the `html:generate` command with the `--output` option, files will be saved here.

Example:
```bash
php artisan html:generate landing-page \
  --output=output/my-page.html \
  --style=full
```

## .gitignore

By default, generated HTML files in this directory are not tracked by Git. If you want to commit specific generated files, you can force add them:

```bash
git add -f output/important-page.html
```

## Organization

Consider organizing your output files:

```
output/
├── landing-pages/
│   ├── clientbridge.html
│   └── smbgen.html
├── blog/
│   └── post-template.html
└── products/
    └── showcase.html
```
