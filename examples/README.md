# Examples

This directory contains example outputs generated by the AI Web Design Workbench.

## Generating Examples

### Landing Page - Full Styling

```bash
php artisan html:generate landing-page \
  --company="ClientBridge" \
  --headline="Ship Faster, Build Better" \
  --subheadline="The developer-first platform for modern web applications" \
  --sections=hero,features,cta,footer \
  --style=full \
  --output=examples/landing-full.html
```

### Landing Page - Mid Styling

```bash
php artisan html:generate landing-page \
  --company="StartupCo" \
  --headline="Simplify Your Workflow" \
  --sections=hero,features,cta \
  --style=mid \
  --output=examples/landing-mid.html
```

### Landing Page - Low Styling

```bash
php artisan html:generate landing-page \
  --company="Simple Solutions" \
  --headline="Clean and Focused" \
  --sections=hero,cta \
  --style=low \
  --output=examples/landing-low.html
```

## Comparing Outputs

Open the generated files in your browser to compare the different styling levels and see how the curated Tailwind classes create different visual densities.
